## Computational Modeling and Simulation Projects
> These projects are a small sample of the work conducted for the course CS346- Computational Modeling and Simulation at Colby College.
> The projects in this course were coded entirely in MATLAB. **All code was written by hand without the use of generative AI**.

### Files:
#### `wfheis26_tmklin26_HW3_exercise5.m`
>  This simulation file performs a simulation of a single neuron's action potential spike based on mathmatical equations for ion flow and voltage-gated channel activation.

#### `bbullo27_jereil26_wfheis26_tmklin26_cs346_hw4_extension1.m`
>  This simulation file performs a cellular automota based simulation of an infectious agent spreading through a population.
>  This simulation uses the states of neighboring cells to update each cell state at every simulation time step.
